Hexo实时预览Markdown

发表于 2019-02-26 更新于 2020-08-25 分类于 Hexo Valine：

用Hexo写文章时因为时用的Markdown语法写的，所以就非常需要进行实时的编辑预览效果，一些文本编辑软件或许可以预览，但是有时和实际总是用点差距，所以介绍的这款插件就可以实现实时预览效果。

Hexo-Browsersync

首先进入Hexo项目目录下，运行如下命令安装hexo-browsersync插件

npm install hexo-browsersync --save

等待安装完成，这个安装过程最好挂全局代理，否则可能有点慢。
然后运行hexo s出现一下内容说明成功啦

$ hexo s
INFO  Start processing
[Browsersync] Access URLs:
 ----------------------------------
          UI: http://localhost:3001
 ----------------------------------
 UI External: http://localhost:3001
 ----------------------------------
INFO  Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.

现在就可以啦，只要在编辑文档时保存一下，网页就会自动进行刷新，就可以马上看到效果啦。
