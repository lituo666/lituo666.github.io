Termux_笔记

发表于 2019-02-15 更新于 2020-08-25 分类于 Termux-笔记 Valine：

基本命令

pkg命令

pkg search <query>              搜索包
pkg install <package>           安装包
pkg uninstall <package>         卸载包
pkg reinstall <package>         重新安装包
pkg update                      更新源
pkg upgrade                     升级软件包
pkg list-all                    列出可供安装的所有包
pkg list-installed              列出已经安装的包
pkg shoe <package>              显示某个包的详细信息
pkg files <package>             显示某个包的相关文件夹路径

目录环境结构

files
|--home
|--usr
   |--bin
   |--lib
   |--share
   |--var
   |--etc
   |--libexec
   |--tmp
$ echo $HOME
/data/data/com.termux/files/home
$ echo $PREFIX
/data/data/com.termux/files/usr
$ echo $TMPPREFIX
/data/data/com.termux/files/usr/tmp/zsh

更换国内源

更换Termux为清华大学源,加快软件包下载速度。

官方推荐方法

设置默认编辑器

$ export EDITOR=vim

编辑源文件

$ apt edit-sources

将原来的 https://termux.net 官方源替换为 http://mirrors.tuna.tsinghua.edu.cn/termux
更换源后需要更新源

$ pkg update

直接编辑源文件

其实还有更简单的方法,类似于Linux下直接去编辑源文件:

$ vim  $PREFIX/etc/apt/sources.list
$ pkg update

安装基本工具

$ pkg update                                    更新源
$ pkg install vim curl wget git unzip unrar     一些基本工具

Termux优化

终端配色

主要使用了oh-my-zsh来替代bash作为默认shell。使用一键安装脚本来安装,一步到位,顺便启动了外置存储,可以直接访问SD卡下的目录。

$ sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)"

分别选择背景色和字体，可以参考如下选择：

Enter a number, leave blank to not to change: 14
Enter a number, leave blank to not to change: 6

想要继续更改挑选配色的话,继续运行脚本来再次筛选：

$ ~/termux-ohmyzsh/install.sh

输入命令exit再回车确认，重启Termux配置生效。

访问外置储存优化

执行过上面的zsh一键配置脚本后,并且授予文件访问权限的话,会在家目录生成storage目录，并且生成若干目录，软连接都指向外置存储卡的相应目录。
为了方便文件传输,直接在storage目录下创建软链接。
创建文件夹软连接
QQ

$ ln -s /data/data/com.termux/files/home/storage/shared/tencent/QQfile_recv QQ

TIM

$ ln -s /data/data/com.termux/files/home/storage/shared/tencent/TIMfile_recv TIM

微信

$ ln -s /data/data/com.termux/files/home/storage/shared/tencent/MicroMsg/Download Wechat

oh-my-zsh主题配色

编辑.zshrc配置文件

$ vim ~/.zshrc

第一行ZSH_THEME="agnoster"，默认是agnoster主题。
在~/.oh-my-zsh/themes目录下放着oh-my-zsh所有的主题配置文件。
下面几款主题不错
agnoster

robbyrussell

jaischeema

re5et

junkfood

cloud

可以尝试random主题,每打开一个会话配色主题都是随机的。

修改启动问候语

编辑问候语文件直接修改问候语：

$ vim $PREFIX/etc/motd

管理员身份

手机没有root

利用proot工具来模拟某些需要root的环境。

$ pkg install proot

然后终端下面输入：

$ termux-chroot

即可模拟root环境
在这个proot环境下面,相当于是进入了home目录,可以很方便地进行一些配置。

在管理员身份下，输入exit可回到普通用户身份。

安装Nodejs

$ pkg install nodejs

如果报错如下：

Cannot read property 'length' of undefined

修改farm.js

$ vim $PREFIX/lib/node_modules/npm/node_modules/worker-farm/lib/farm.js

尝试将length:后指定一个数字，例如length:4

安装Hexo博客

$ npm install hexo-cli -g
$ hexo init Blog
$ cd Blog
$ npm install
